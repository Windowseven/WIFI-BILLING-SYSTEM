### **Step 1: Find the Network**
1. Open WiFi settings on your device
2. Look for network name: **"Windowseven_Hotspot"**

## **🔧 Troubleshooting**

### **Can't Connect to WiFi:**
1. **Check network name:** "Windowseven_Hotspot"
2. **Restart WiFi:** Turn off/on WiFi on your device
3. **Forget & reconnect:** Remove network, connect again
4. **Move closer:** Get closer to the hotspot

### **Not Redirected to Login Page:**
1. **Try different website:** Visit google.com or facebook.com
2. **Clear browser cache:** Delete browsing data
3. **Try different browser:** Chrome, Safari, Firefox
4. **Manual access:** Go to `http://10.10.0.1:8080`

### **Voucher Code Not Working:**
1. **Check code format:** Should be WIFI-2024-XXX
2. **Try again:** Re-enter the code carefully
3. **Code expired:** Check with vendor
4. **Already used:** Each code works only once

### **Slow Internet:**
1. **Check data remaining:** You might be near limit
2. **Restart device:** Turn off/on your device
3. **Move location:** Get closer to hotspot
4. **Peak hours:** Try again later

### **Session Stopped Unexpectedly:**
1. **Check time/data:** Session might have expired
2. **Reconnect to WiFi:** Connection might have dropped
3. **Browser refresh:** Reload the page
4. **Contact support:** If problem persists
