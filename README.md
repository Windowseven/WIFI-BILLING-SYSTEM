# WiFi Billing System 📶

A web-based WiFi billing and management system designed for small ISPs, hostels, hotels, and internet cafes in Tanzania and across East Africa.

## Problem It Solves

Small internet providers in East Africa often manage client billing manually — spreadsheets, WhatsApp messages, guesswork. This system automates session tracking, billing, and client management through a clean web interface.

## Features

- 👤 Client registration and profile management
- 💰 Billing and payment tracking
- ⏱️ Session time management
- 📊 Usage reports and analytics dashboard
- 🔐 Admin authentication and access control
- 📱 Responsive UI for desktop and mobile

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML5, CSS3, JavaScript |
| Styling | Custom CSS + Bootstrap |
| Backend | Node.js / PHP |
| Database | MySQL |

## Getting Started

```bash
git clone https://github.com/Windowseven/WIFI-BILLING-SYSTEM.git
cd WIFI-BILLING-SYSTEM

# If using a local server (e.g. XAMPP/WAMP)
# Copy to htdocs and open in browser
# OR
npm install && npm start
```

## Target Market

Built specifically for the Tanzanian and East African market where affordable internet management tools are in high demand.
